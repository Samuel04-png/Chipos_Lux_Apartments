# Chipo's Lux Apartments Images

The live website now uses optimized image copies in `public/images/site`.

The original apartment photo files can stay in this folder. To change the live selection, update the image paths in `src/content.ts`, then replace or regenerate the matching files in `public/images/site`.

Original starter placeholder names were:

- `hero-apartment.jpg`
- `bedroom-1.jpg`
- `living-room.jpg`
- `kitchen.jpg`
- `bathroom.jpg`
- `exterior.jpg`
- `apartment-detail.jpg`

Keep the logo as `chipos-logo.png` unless the business provides an updated logo file.
